# vue-touch

`vue create vue-touch`  merge  
`$ cd vue-touch`  
`$ yarn serve`  

npm [ Vue 2.0 移动设备指令( Vue touch directive )](https://www.npmjs.com/package/vue-directive-touch)  

[vue-directive-touch事件類型](https://www.helloweba.net/javascript/610.html)：

單擊： v-touch:tap  
長按： v-touch:long  
左滑： v-touch:left  
右滑： v-touch:right  
上滑： v-touch:up  
下滑： v-touch:down
